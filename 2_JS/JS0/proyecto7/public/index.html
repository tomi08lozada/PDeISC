indexOf()

🐶 Buscar "perro"

Encuentra la posición de la palabra "perro" en un array de animales.

const animales = ['gato','perro','conejo','perro'];
const pos = animales.indexOf('perro');
// pos → 1 (primera ocurrencia)
// Usa === para comparar (estricto)
🔢 ¿Está el número 50?

Verifica si el número 50 está en el array y en qué posición.

const nums = [10,30,50,70,90];
const idx = nums.indexOf(50);
if (idx !== -1) {
  console.log('Encontrado en:', idx);
} else {
  console.log('No está');
}
🌍 Buscar ciudad

Muestra el índice de "Madrid" en el array de ciudades, o un mensaje si no está.

const ciudades = ['París','Madrid','Roma','Berlín'];
const idx = ciudades.indexOf('Madrid');
// idx !== -1 ? mostrar índice : "No encontrado"
