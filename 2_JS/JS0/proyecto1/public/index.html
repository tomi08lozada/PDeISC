Array.push()

Agrega uno o más elementos al final del array y retorna la nueva longitud.

🍎 Array de frutas vacío

Crea un array vacío y agrega tres frutas usando push() una por vez.

const frutas = [];
frutas.push('Manzana');
frutas.push('Banana');
frutas.push('Naranja');
👥 Agregar 3 amigos

Agrega los nombres de 3 amigos a un array existente con un solo push() de múltiples argumentos.

const amigos = ['Carlos'];
// push acepta múltiples argumentos
amigos.push('Lucía', 'Mateo', 'Sofía');
🔢 Push condicional

Agrega un número solo si es mayor que el último elemento del array.

// arr[arr.length - 1] = último elemento
if (nuevo > arr[arr.length - 1]) {
  arr.push(nuevo);
}
