Array.unshift()

Agrega uno o más elementos al principio del array y retorna la nueva longitud.

🎨 Agregar colores al inicio

Agrega tres colores al principio de un array vacío usando unshift().

const colores = [];
colores.unshift('Azul');
colores.unshift('Rojo');
colores.unshift('Verde');
// → ['Verde','Rojo','Azul']
⚡ Tarea urgente primero

Dado un array de tareas, agrega una nueva tarea urgente al principio.

const tareas = ['Estudiar','Leer','Hacer gym'];
tareas.unshift('🚨 URGENTE: Entregar proyecto');
👤 Usuario conectado primero

Inserta el nombre de un usuario al principio del array de usuarios conectados.

const usuarios = ['Ana','Luis','Pedro'];
usuarios.unshift(nuevoUsuario);
// El nuevo aparece primero
