Array.shift()

🔢 Quitar primer número

Quita el primer número de un array de enteros con shift().

const numeros = [10, 20, 30, 40, 50];
const primero = numeros.shift();
// primero → 10
// numeros → [20, 30, 40, 50]
💬 Eliminar primer mensaje

Elimina el primer mensaje de un array de chat con shift().

const chat = ['Hola!','¿Cómo estás?','Bien, gracias'];
const leido = chat.shift();
// leido → 'Hola!'
🏪 Cola de atención al cliente

Simula una cola FIFO: los clientes se atienden en orden de llegada con shift().

// push() → entra al final de la cola
// shift() → se atiende el primero
cola.push(nuevoCliente);
const atendido = cola.shift();
