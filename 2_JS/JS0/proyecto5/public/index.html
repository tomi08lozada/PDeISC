splice()

✂️ Eliminar desde posición 1

Elimina dos elementos comenzando desde la posición 1 de un array de letras.

const letras = ['a','b','c','d','e'];
// splice(inicio, cantEliminar)
const eliminados = letras.splice(1, 2);
// eliminados → ['b', 'c']
// letras → ['a', 'd', 'e']
➕ Insertar sin eliminar

Inserta un nuevo nombre en la segunda posición sin eliminar ningún elemento.

const nombres = ['Ana','Luis','Pedro'];
// cantEliminar = 0 → solo insertar
nombres.splice(1, 0, 'Sofía');
// → ['Ana','Sofía','Luis','Pedro']
🔄 Reemplazar elementos

Reemplaza dos elementos por otros nuevos desde una posición determinada.

const frutas = ['Manzana','Pera','Uva','Kiwi'];
// Reemplaza 2 desde pos 1
frutas.splice(1, 2, 'Mango', 'Fresa');
// ['Manzana','Mango','Fresa','Kiwi']
