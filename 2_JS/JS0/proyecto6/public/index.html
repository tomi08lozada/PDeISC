slice()

🔢 Primeros 3 elementos

Copia los primeros 3 elementos de un array de números sin modificar el original.

const nums = [10,20,30,40,50,60];
// slice(0, 3) → desde 0, hasta 3 (sin incluir)
const primeros = nums.slice(0, 3);
// → [10, 20, 30]
// nums original no cambia
🎬 Copia parcial de películas

Crea una copia parcial del array de películas desde la posición 2 hasta la 4.

const pelis = ['Matrix','Avatar','Inception',
               'Interstellar','Dune','Oppenheimer'];
// slice(2, 5) → índices 2, 3, 4
const seleccion = pelis.slice(2, 5);
🔚 Últimos 3 elementos

Crea un array con los últimos 3 elementos usando índice negativo.

const arr = ['a','b','c','d','e','f'];
// Índice negativo: cuenta desde el final
const ultimos = arr.slice(-3);
// ['d', 'e', 'f']
