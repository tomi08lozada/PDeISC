Array.pop()

🐾 Eliminar último animal

Elimina el último elemento de un array de animales con pop().

const animales = ['perro','gato','pájaro','pez'];
const eliminado = animales.pop();
// eliminado → 'pez'
🛒 Lista de compras

Quita el último producto de la lista y muestra cuál fue eliminado.

const lista = ['leche','pan','arroz','aceite'];
const quitado = lista.pop();
console.log('Eliminado:', quitado);
🔄 Vaciar con while

Usa un bucle while para vaciar completamente un array usando pop().

while (arr.length > 0) {
  const item = arr.pop();
  // procesar item...
}
